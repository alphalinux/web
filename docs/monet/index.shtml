Monet

Monet (XP1000) systems are EV6 (21264) based system with 1 CPU and 2 PCI buses. Compaq has recently started supporting Linux on XP1000.

Note: The kernel that ships with RedHat 5.2 (the DP264 kernel) will not work on the Monet. Use the kernel image below. Also note that a network install does not seem to work at the moment, so loading from a local CD-ROM is the best choice.

XP1000 machines must be started from SRM as there is currently no MILO image.
