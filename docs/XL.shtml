XL

The XL machine is based on the 21064 CPU and runs either at 233 or 266MHz with 512KB or 2MB of second level cache, respectively. The newer XL Turbo is based on the 21164A CPU and runs either at 300 or 366MHz. The Turbo always comes with 2MB of second-level cache.

    • Warning for the XL-233:
    There is a problem with the ISA-PCI bridge on this machine (and possible others ?) that makes it impossible to boot a kernel from the floppy drive. Instead Linux should be booted from a SCSI device. Note that this problem does not seem to effect the XLT (300 and 366MHz machines).
