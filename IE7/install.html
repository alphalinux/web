Installation (simplest)

Follow these simple instructions to get IE7 working immediately on your server:

  • download either ie7.zip or ie7-full.zip

  • unzip the entire contents to the root of your server (keep the folder names used in the zip file)

  • you will now have an IE7 directory in the root of your server

  • include the IE7 style sheet in the page you wish to test

    <!-- compliance patch for microsoft browsers -->
    <!--[if lt IE 7]>
    <link rel="stylesheet" href="/IE7/ie7-html.css" type="text/css"/>
    <![endif]-->

  • make sure this also points to the same "IE7" directory

  • open the page in your web browser

  • the page should now be IE7 enabled.

You may also extract the contents of the zip file to the root of your hard disk if you do not have access to a web server.

If you downloaded the full installation you can use the IE7 compatibility suite on your own server.

Troubleshooting (basic)

If things are not as expected try doing the following:

  • in a text editor, open the file ie7-html.htc

  • locate the reference to the module ie7-debug.js

  • remove the comments surrounding this reference

  • reload the test page

You should get an alert for every module as it loads. You will also get a message if an error occurs. If problems persit you can visit the IE7 forum.
