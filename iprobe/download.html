 Use Iprobe to find performance bottlenecks.  Make your alpha faster. 

READ ME FIRST:

  1. The iprobe suite runs on all Alpha machines prior to the EV6. 
  2. Iprobe also requires a kernel >2.1.132, including all  2.2.X kernels. (Redhat 6.0 should work) 
  3. It has been tested successfully on all Alpha/Linux machines that boot using the Digital SRM and aboot. 
  4. It  is NOT verified to work on machines running MILO.  Your milage may vary! 
READ ME SECOND:

This installation guide will tell you how to install iprobe on your Alpha/Linux machine.
