For lack of a better name. That's what I'm calling it for now. Not much code written thus far. Way I tend to work is planning 70% of the time and the rest is coding. A good architechture that is well documented can provide developers (myself included) a nice scafolding to work in. For those who are not on the major Alpha lists. I am creating a new firmware system for Alphas derived from milo with a feature set that will exceed that of SRM in some places while leaving out the things that only developers need. I hope to get access from Compaq an updated binary of the BIOS emulator for this project.
