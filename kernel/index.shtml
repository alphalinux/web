    Obtaining Kernel Source
    The Linux Kernel Archive Mirror System is set up so that for each two-letter country code from the list below, you can simply use the host names www.country-abrivation(us,jp etc.).kernel.org or ftp.country-abrivation.kernel.org to reach a mirror supporting that specific country. Each mirror will have a full archive of /pub/Linux/ and /pub/software/, although they may not carry both the gzip and bzip2 compression formats. For a list of these sites please visit http://www.kernel.org/mirrors/.
