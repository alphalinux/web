SuSE Update for kreatecd
Wed Apr 5 16:21:34 2000
  SuSE has released an update for the kreatecd package due to an exploitable buffer overflow that could lead to local users getting root authority. An update for SuSE 6.4 has been posted.

SuSE Security Update for GPM
Wed Apr 5 16:11:12 2000
  SuSE has issued a security update for gpm for the following reason: "The gpm-root command, which is part of the gpm package, allows local users to define menus and commands to be executed on mouse events. When a command is executed via gpm, the group id 0 privilege is not dropped." Updates are available from ftp.suse.com for SuSE AXP 6.1 and SuSE AXP 6.3.

Mandrake for Alpha
Mon Apr 3 17:24:40 2000
  Linux-Mandrake 7.0 for Alpha is finally available. Like the UltraSPARC version released last week, Mandrake 7.0 for Alpha is a complete port (graphical installation included) of Linux-Mandrake 7.0/x86-final, but could still have some bugs. It is not intended for production use yet.

Download on http://www.linux-mandrake.com/en/ftp.php3#alpha

Bugs reports should be sent to the Cooker list on www.linux-mandrake.com/en/fdevlists.php3
