Old News....

Warning links in here may no longer be valid and they will not be updated.

May 28th 1999
  RedHat has released more updates for RedHat 6.0 and 5.2 on Alpha. There are some problems with the PHP3 module shipped with 6.0 and the mod_perl package from 5.2 will not function properly with the latest errata version of Perl(perl-5.004m7-1). Updates are available from ftp.alphalinux.org.

May 27th 1999
  Blender version 1.63 is now available for AlphaLinux from the Blender Download page.

May 25th 1999
  RedHat has released a new INN package due to security problems with the original package shipped with 6.0. The updated RPMs can be picked up from ftp.alphalinux.org.

May 24th 1999
  MetroLink has ported their hardware accelerated OPENGL product to AlphaLinux.

TurboLinux is in the process of producing an Alpha distribution. The now brings the total number of Alpha distributions to four (RedHat Debian SuSE TurboLinux).

May 24th 1999
  AlphaLinux binaries for the SETI@home project are available from the SETI@home Download Page.

"SETI@home is a scientific experiment that will harness the power of hundreds of thousands of Internet-connected computers in the Search for Extraterrestrial Intelligence (SETI). You can participate by running a program that downloads and analyzes radio telescope data. There's a small but captivating possibility that your computer will detect the faint murmur of a civilization beyond Earth."

May 19th 1999
  Leslie Donaldson has begun to port and extend the EM86 emulator to work with the 2.2.x kernels. See his post on comp.os.linux.alpha and his homepage for more information about the project.
May 19th 1999
  Alpha Processor Inc. announced today that it has joined Linux International. The official press release can be at LinuxToday.
May 18th 1999
  Blender version 1.62a is now available for AlphaLinux from the Blender Home Page.
May 17th 1999
  Cygnus has cut a deal with Alpha Processor Inc. to port the GNUPro Toolkit to AlphaLinux. The original press release can be found at LinuxToday . Availability is set for 3rd quarter 1999.
