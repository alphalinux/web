Netscape RPM Removed
Thu Jan 27 11:36:36 2000
  As of 11:30AM EST today (Jan 27th, 2000) the Netscape RPM has been removed from ftp.alphalinux.org due to this email from Compaq. If you want to download the package you must now register via the Compaq site. As can be seen, Compaq is claiming that they must track the number of downloads; however, when ALO offered to also track this number for them, the request was ignored.

Netscape For AlphaLinux!
Tue Jan 25 11:44:48 2000
  Compaq has packaged together Netscape for Tru64 Unix along with necessary Tru64 libraries into an RPM for AlphaLinux. The RPM can be downloaded from either ftp.alphalinux.org or empeor.alpha-processor.com. Remember that if you compile your own kernel you need a.out (ECOFF) support in the kernel for this to work.
