 3/14/2000:

Hello!  This is a really sparse page right now, but I would like it to become a rallying point for all things Alpha/LInux & gaming. 

Things I am planning for the future:

  1. Hook up with the Alpha/Linux glide people, and help port glide.  
  2. Help demo writers port to Alpha.
  3. Distribte game binaries compiled with the Compaq compilers & some decent optimzation switches. 
  4. Produce a Alpha optimzation guide for game writers.
  5. Review Commercial Alpha/Linux games.
  6. Create an Alpha/Linux Gaming FAQ
Ultimately, I think that a successful Alpha/Linux  is one where all opernsource programs compile out of the box on Alpha/Linux, so all patches will be contributed back to the project leads. 

If you have comments, additions,  or game review suggestions,  email me at Phillip.Ezolt@compaq.com
