Howto mirror alphalinux.org


In order to mirror alphalinux.org we require that you first register with ALO. Currently mirroring is done via rsync. In order to mirror alphalinux.org please send the following information to admin@alphalinux.org

  • Contact Name and Email Address
  • Geographic Location
  • Machine Name and IP Address
  • Local Directory where the files will be
  • Bandwidth Available
  • Disk Space Available

We also require that the administrator join the ALO mirror list. To subscribe send an email to majordomo@alphalinux.org with the text 'subscribe alo-mirror' in the body. The size of the ftp archive is currently about 1.5Gig in size. We do not at this point allow mirroring of the distributions from alphalinux.org.
